# Semantic Resume Screening and Ranking System using BERT

## 📌 Overview
This project is an AI-powered system that performs semantic resume screening and ranking based on job descriptions. It leverages BERT embeddings to understand contextual meaning and improve matching accuracy compared to traditional keyword-based approaches.

---

## 🚀 Features
- 🔍 Semantic similarity using BERT (Sentence Transformers)
- 📊 Section-wise analysis (Skills, Experience, Projects)
- 📈 Resume ranking based on weighted scoring
- 🧠 Hybrid approach (BERT + TF-IDF + Skill Mapping)
- 📄 PDF report generation
- 🎯 Skill gap analysis (Matched & Missing Skills)

---

## 🧠 Tech Stack
- Python
- Flask
- Sentence Transformers (BERT)
- Scikit-learn (TF-IDF, Cosine Similarity)
- HTML/CSS (Frontend)

---

## ⚙️ How It Works
1. Extract text from resumes
2. Preprocess and split into sections
3. Compute semantic similarity using BERT
4. Calculate weighted scores
5. Perform skill matching and coverage analysis
6. Rank resumes and generate report

---

## 📊 Scoring Logic
- Skills: 50%
- Experience: 30%
- Projects: 20%

---

## 🧪 Results
- Improved matching accuracy using semantic understanding
- Better handling of synonyms and contextual meaning
- Reduced dependence on exact keyword matching
